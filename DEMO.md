# ASD儿童饮食与营养均衡系统 - 演示指南

## 🎯 演示目标

展示ASD儿童饮食与营养均衡系统的核心功能和用户体验，包括：

## 📋 演示准备

### 环境检查

```bash
# 1. 启动后端服务
cd asd
./mvnw spring-boot:run

# 2. 启动前端应用（新终端）
cd frontend
npm run dev

# 3. 确认服务状态
# 后端: http://localhost:8081 (API)
# 前端: http://localhost:8080 (用户界面)
```

## 🎬 演示流程

### 第一部分：系统介绍 (2分钟)

#### 1.1 项目背景
```
ASD儿童饮食与营养均衡系统是一个专门为自闭症谱系障碍儿童
设计的营养管理平台，帮助家长和营养师：

• 记录和管理儿童饮食信息
• 跟踪营养摄入情况
• 提供专业营养建议
• 生成个性化营养报告
```

#### 1.2 技术架构
```
• 后端: Spring Boot 2.7 + Java 8 + JPA
• 前端: Vue 2 + Element UI + Vuex
• 数据库: H2 (开发) / MySQL (生产)
• 安全: Spring Security + JWT
```
